## About This Project

The Australian Child Support Calculator is an open-source web application that helps Australian parents understand and calculate child support obligations according to the official Department of Social Services (DSS) guidelines.

### Key Features

- **Official Formula Implementation** - Complete 8-step Australian child support calculation
- **Privacy First** - All calculations happen locally in your browser
- **Educational Focus** - Interactive guides explain each step
- **Modern Design** - Professional neumorphic interface
- **Offline Capable** - Works without internet after initial load

### Technology Stack

- React 18 with TypeScript
- Vite for fast builds
- Tailwind CSS for styling
- Radix UI for accessible components
- Local storage for data persistence
